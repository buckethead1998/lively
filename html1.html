1. Introduction to HTML

1.1. Struture Text

Horizontal line The < hr > element is used to separate content (or define a change) in an HTML page.
The < hr > tag is an empty tag which means that it has no end tag.

Line Break The < br > tag inserts a single line break.
The < br > tag is an empty tag which means that it has no end tag.

Headings Headings are defined with the < h1 > to < h6 > tags.

< h1 >Heading 1< /h1 >

Heading 1


< h2 >Heading 2< /h2 >

Heading 2


< h3 >Heading 3< /h3 >

Heading 3


< h4 >Heading 4< /h4 >

Heading 4


< h5 >Heading 5< /h5 >

Heading 5

< h6 >Heading 6< /h6 >

Heading 6

Emphasized text The < em > tag is a phrase tag. It renders as emphasized text.

1.2. HTML Lists

< li >Item 2< li >
< li >Item 3< li >
< /ol >

  2. Item 2
  3. Item 3

1.4. Anchor Tag Download Attribute

Hyperlink
The < a > tag defines a hyperlink, which is used to link from one page to another.
The most important attribute of the < a > element is the href attribute, which indicates the link's destination.

< a href="#">Hyperlink< /a>

Result:
Hyperlink

Additional:
Download Attribute
< a download="#" href="#">< /a>

Result:
Download link

New window or tab
< a href="#" target="_blank">< /a>

Result:
Open hyperlink in new window or tab

2. Intermediate HTML

1.1. HTML Tables

Table tag The < table > tag defines an HTML table.
An HTML table consists of the < table > element and one or more < tr >, < th >, and < td > elements.


< table border="3" bgcolor="red" cellpadding="8" cellspacing="5" width="30%" >
< thead >
< tr >
< th >Tomislav < /th >
< th >Mladen < /th >
< /tr >
< /thead >
< tbody >
< tr >
< td >1998< /td >
< td >1995< /td >
< /tr >
< /tbody >
< tfoot >
< tr >
< td >Elektricar< /td >
< td >Varioc< /td >
< /tr >
< /tfoot >
< /table >

Result:

Tomislav Mladen
1998 1995
Elektricar Varioc

1.2. HTML Forms

Form tag The < form > tag is used to create an HTML form for user input.
The < form > element can contain one or more of the following form elements:

  • < input >
  • < textarea >
  • < button >
  • < select >
  • < option >
  • < optgroup >
  • < fieldset >
  • < label >
  • < output >
